# Youtube-video-Summarization
Speech to Text Summarization of Youtube Video URL using Transformers Pipeline Function
This project provides a Python-based solution to summarize YouTube videos by extracting their transcripts and applying a pre-trained summarization model from the Hugging Face transformers library.

Features
Transcript Extraction: Automatically fetches transcripts from YouTube URLs using youtube-transcript-api.
AI-Powered Summarization: Utilizes the facebook/bart-large-cnn model for generating concise summaries of video content.
Easy to Use: Designed for straightforward execution in environments like Google Colab.
